# 🚀 Galaxy Defender

A modern 2D Space Shooter game built using **React Native** and **Expo**.

🌐 **Live Demo:** https://galaxy-defender-alpha.vercel.app

## 🎮 Gameplay

- Control your spaceship
- Destroy enemy ships
- Defeat powerful bosses
- Collect coins
- Unlock achievements
- Complete missions
- Survive as long as possible

---

## ✨ Features

- 🚀 Smooth spaceship movement
- 🔫 Laser shooting
- 👾 Multiple enemy types
- 💥 Collision detection
- ❤️ Lives system
- 🏆 Score & High Score
- 📈 Level progression
- 👑 Boss battles
- 🪙 Coin collection
- ⚡ Power-ups
- 🎯 Missions
- 🛍️ Shop System
- 🏅 Achievements
- ⏸️ Pause & Resume
- 🌌 Animated Star Background
- 📱 Mobile & Web Support

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/rishi4842/GalaxyDefender.git
```

### Go into the project

```bash
cd GalaxyDefender
```

### Install dependencies

```bash
npm install
```

### Start the project

```bash
npx expo start
```
